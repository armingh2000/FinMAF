# Spark Tests

## Description
This directory contains the source codes for the whole end-to-end system.

## File Structure
- **conftest.py**:  
  Contains configurations for testing environments, setting up fixtures for the Pytest framework.
- **test_data_download.py**:  
  Includes unit tests for data download functionalities, ensuring data is fetched correctly.
- **test_data_processing.py**:  
  Contains tests for data processing functions, verifying the integrity and correctness of data transformations.
