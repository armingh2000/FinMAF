# Spark Data Download and Processing using Apache Spark

## Description
This directory contains code for downloading and cleaning the stock data used for training the model. Configs for this directory are set in src/configs/data_pipeline_spark.py file.

## File Structure
- **data_download.py**: Handles the downloading of data from specified sources.
- **data_processing.py**: Contains functions for processing and cleaning the downloaded data.
- **main.py**: The main script that runs the application, orchestrating the data flow and execution logic.
- **utils.py**: Provides utility functions or classes used across the project.


## Usage
python3 main.py
