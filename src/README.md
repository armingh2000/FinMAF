# Src

## Description
This directory contains the source codes for the analysis system.

## File Structure
- **data_pipeline/**:
Contains the codes for Apache Spark (offline training) and Apache Kafka (online training) pipelines.
- **configs/**:
Contains configuration codes for the rest of the project.
- **log/**:
Contains logging scripts.
- **__init__.py**:
Created for future importing usage.
- **utils.py**:
Contains general usage functions that may be required in any project files.
